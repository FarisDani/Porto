Teras Budaya

A cultural platform for discovering Indonesian heritage and events.

Main Features

  • Browse Indonesian cultural events and traditions
  • Filter events by region or category
  • Interactive cultural map for exploration
  • Save favorite events for quick access
  • Multilingual support (English & Bahasa Indonesia)

Tools & Technologies

HTML5 Tailwind CSS JavaScript Figma (UI/UX) JSON API Git & GitHub
